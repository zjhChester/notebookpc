绑定域名后，访问app端将使用绑定的域名，且绑定的域名只对手机端生效；

如访问微站链接为{$_W['siteroot']}app/index.php?i={$_W['uniacid']}&c=home，
    绑定bang.baidu.com后，访问微站链接就成为http://bang.baidu.com/app/index.php?i={$_W['uniacid']}&c=home

绑定域名，只支持一级域名和二级域名；

绑定域名后，请将公众平台的业务域名、js接口安全域名、网页授权域名都改为您绑定的域名，
    如本说明中绑定bang.baidu.com，则改为bang.baidu.com；

绑定域名后，服务器配置中的服务器地址（URL）不变，仍为：{$_W['siteroot']}api.php?id={$_W['uniacid']}

还有最后很重要的一步：将绑定的域名解析到本服务器IP并绑定到微擎网站目录。

此功能不支持授权接入的公众号。
